## My First Nest App

This project is based on [Let’s build a REST API with NestJS and Prisma - Tasin Ishmam I Prisma Day 2022](https://www.youtube.com/watch?v=LMjj1_EK4y8) by [Prisma](https://www.youtube.com/@PrismaData)

[Step-by-Step text guide](https://prismaio.notion.site/1-Set-up-Prisma-PostgreSQL-and-NestJS-b2b03341a7cf4df0a6f77e7107be0365)

### Commands

```sh
  # Create a new NestJS project
  $ npx @nestjs/cli new myfirstnestapp


  cd myfirstnestapp

  npm run start
```

### Create docker-compose file

```sh
  touch docker-compose.yml
```

### Install and setup Prisma

```sh
  npm install prisma --save-dev

  npx prisma init

  npx prisma migrate dev --name init

  # Create seed file
  touch prisma/seed.ts

  # Create seed script on package.json
  "prisma:seed": "ts-node prisma/seed.ts"

  # Run seed script
  npx prisma db seed

  # View data on Prisma Studio
  npx prisma studio

  # Initialize Prisma Module and Service
  npx nest generate module prisma
  npx nest generate service prisma
```

### Setup Swagger

```sh
  npm install --save @nestjs/swagger swagger-ui-express
```

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
